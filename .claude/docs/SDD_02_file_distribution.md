## 1. 개요

### 1.1 목적
파일 패키지를 생성하고 Edge 디바이스에 배포하는 기능을 설계한다.

### 1.2 범위
- 배포 패키지/버전 관리
- 배포 파일 업로드
- 배포 실행 (즉시/예약)
- 승인 워크플로우
- 배포 대상 지정 및 결과 추적

---

## 2. 기능 설계

### 2.1 기능 목록

| No | 기능명 | 설명 | 우선순위 |
|----|--------|------|----------|
| 1 | 패키지 관리 | 배포 패키지 생성/수정/삭제 | 상 |
| 2 | 버전 관리 | 패키지 버전 관리 | 상 |
| 3 | 파일 업로드 | 배포 대상 파일 업로드 | 상 |
| 4 | 배포 실행 | 즉시/예약 배포 실행 | 상 |
| 5 | 승인 관리 | 배포 실행 승인 워크플로우 | 중 |
| 6 | 대상 지정 | 배포 대상 Edge/그룹 지정 | 상 |
| 7 | 결과 조회 | 배포 결과 및 상태 추적 | 중 |

### 2.2 기능 흐름도

```
[패키지 생성] -> [버전 등록] -> [파일 업로드]
    -> [배포 대상 지정] -> [승인 요청] -> [승인 완료]
    -> [배포 실행] -> [Edge 수신/적용] -> [결과 보고]
```
